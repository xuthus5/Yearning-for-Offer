输入一个链表，输出该链表中倒数第k个结点。

JAVA

```java
public class Solution {
    public ListNode FindKthToTail(ListNode head,int k) {
        if (head == null || head.next == null) return head;
        ListNode front = new ListNode(0);
        front.next = head;
        ListNode end = head;
        for (int i=0;i<k;i++) {
            front = front.next;
            if (front == null) {
                return null;
            }
        }
        while (front.next != null) {
            front = front.next;
            end = end.next;
        }
        return end;
    }
}
```
