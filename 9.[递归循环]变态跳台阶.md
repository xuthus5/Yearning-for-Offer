JAVA

```java
public class Solution {
    public int JumpFloorII(int target) {
        if (target <= 1) {
            return 1;
        }
        return 2*this.JumpFloorII(target-1);
    }
}
```

存在公式推导如下：
$$
f(n)=f(n-1)+f(n-2)+f(n-3)+...+f(n-n)
$$

$$
f(n-1)=f(n-2)+f(n-3)+f(n-4)+...+f(n-n)
$$

$$
(1)+(2)=f(n)=2[f(n-2)+f(n-3)+f(n-4)+...+f(n-n)]=2f(n-1)
$$
